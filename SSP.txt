.SSB - SteveScratch Project File Structure

用于方便 SSC 的工程存读取

## 符号解释

{} = 以上一个整数为计次循环读取
"" = 字符串（NULL可有可无）
"*" = 任意以 NULL 结尾的字符串
"b" = 任意的字符串或二进制数据（NULL可有可无）
(#num) = 有部分说明，请查阅注解段
uint = 无符号整数
luint = 无符号长整数
double = 双精度浮点数

!(符号) = 该符号在混淆工程内不会出现

## 注解

1. 标识符有如下：
	* 0b1	此工程为已混淆工程
	* 0b10	此工程为 SSCN 工程

2. 数据将会在程序内部按读取顺序编号，以留后面读取引用

3. 舞台属于 0 号角色，其余的角色按读取顺序排序编号
